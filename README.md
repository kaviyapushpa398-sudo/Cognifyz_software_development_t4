# 🌡️ Temperature Converter (Python)

## 📌 Overview

This project is a **Temperature Converter** built using Python. It allows users to convert temperatures between **Celsius and Fahrenheit** through an interactive console-based interface.

---

## 🎯 Objective

To develop a user-friendly application that performs temperature conversions accurately while providing an engaging user experience.

---

## 🏢 Project Source

This project was completed as part of a task provided by **Cognifyz**.

---

## 🛠️ Features Implemented

### ✅ 1. User Input Handling

* Accepts temperature value from the user
* Allows selection of conversion type:

  * Celsius → Fahrenheit
  * Fahrenheit → Celsius

---

### ✅ 2. Conversion Logic

* Implements correct formulas for:

  * Celsius to Fahrenheit
  * Fahrenheit to Celsius
* Displays accurate and formatted results

---

### ✅ 3. Interactive Menu

* Menu-driven interface
* Allows multiple conversions in a single run

---

### ✅ 4. Clear Output Display

* Displays results in a readable format

  * Example: `25°C = 77°F`

---

### ✅ 5. Conditional Logic

* Uses if-else statements to control program flow
* Handles user choices effectively

---

### ✅ 6. Error Handling

* Handles invalid inputs (non-numeric values)
* Displays user-friendly error messages

---

### ✅ 7. Bonus Features

* Displays temperature category:

  * Very Cold / Cold / Warm / Hot
* Provides small suggestions based on temperature
